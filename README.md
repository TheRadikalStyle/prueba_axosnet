# prueba_axosnet
Prueba para vacante android proporcionada por Axosnet

Librerias utilizadas
- Librerías android
- Retrofit

Permisos
- .INTERNET
- .ACCESS_NETWORK_STATE

Orientación
- Landscape
- Portrait

Lenguaje
- Español (predeterminado)
- Inglés
(Depende del idioma principal de android)

Versiones compatibles
Desde API 21 a API 30
